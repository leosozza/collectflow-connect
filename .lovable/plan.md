

# Desfazer alterações na tela de Atendimento

Para reverter a tela de atendimento ao estado anterior, você pode usar o recurso de histórico do Lovable.

Clique no botão de "reverter" abaixo da mensagem anterior ao prompt do redesign (a mensagem onde foi implementado o botão WhatsApp e edição de telefones), ou use a aba de Histórico para selecionar a versão desejada.
